Trash Destroyers

Trash Destroyers was a project me and 3 other people worked on that we submitted to HenHacks 2024, which was our first hackathon and which one first place for the Social Good Category that we submitted to. This project involved a website component where users can post about areas with litter in their community and others can respond if they cleaned it up. There was also a game component where users who cleaned up the community was rewarded with in-game currency which could be used to level up equipment to fight monsters. My personal contribution was the Home screen as well as the interaction between the player and the monster on the game, which was done using Pygame.

Booktracker

This Book Tracker was made as a personal project so I can have an online tracker of books that I read where I’m able to give half star ratings. To make this project, I used HTML, CSS, and JavaScript and it was my first time using the React Library. This project is a person booktracker which allows users to add books to a bookshelf. When adding a book, they can add the title, author, rating out of 5 stars, and a review. After adding a book to the shelf, they can click on the book to view and/or edit it.

Scandere

Scandere is a 2 person platformer where both players need to reach the cave entrance in order to win. Each players had 3 hearts and must use a flashlight to scare off incoming bats or else they will get attacked and lose a heart. This project was made using Python and the Designer library.
