Booktracker

This project is a person booktracker which allows users to add books to a bookshelf. When adding a book, they can add the title, author, rating out of 5 stars, and a review. After adding a book to the shelf, they can click on the book to view and/or edit it. This single page application was written in TypeScript, CSS, and HTML and utilized the React library.

Scandere

Scandere is a 2 person platformer where both players need to reach the cave entrance in order to win. Each players had 3 hearts and must use a flashlight to scare off incoming bats or else they will get attacked and lose a heart. This project was made using Python and the Designer library.
